# Larapress Authentication
A package to provide ECommerce & Product management based on Model in Larapress Profiles.

## Dependencies
* Larapress CRUD
* Larapress Reports
* Larapress Profiles

## Install
1. ```composer require ```

## Config
1. Run ```php artisan vendor:publish --tag=larapress-ecommerce```
